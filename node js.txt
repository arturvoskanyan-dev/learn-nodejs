| #   | Թեմա                      | Ինչու է կարևոր                                                                                |
| --- | ------------------------- | --------------------------------------------------------------------------------------------- |
| 1️ Modules: CommonJS     | Սովորում ես `require`, `exports`։ Սկզբից սա պետք է իմանաս, որ կոդը բաժանես մասերի։            |
| 2️ HTTP                  | Սովորում ես server սարքել, request/response հասկանալ։ Բոլոր backend-ների հիմքն է։             |
| 3️ File system (fs)      | Ֆայլ կարդալ/գրել/ջնջել։ Օգտագործվում է ամեն տեղ։                                              |
| 4️ Path                  | Ֆայլերի ուղիներ կառավարելու համար՝ շատ կարեւոր է կայունության համար։                          |
| 5️ Process               | Սովորում ես աշխատել `process.env`, `argv`, `exit` և այլն։ Օգտագործվում է բոլոր պրոյեկտներում։ |
| 6️ Events (EventEmitter) | Սկսում ես հասկանալ async logic, sockets-ի հիմքը։                                              |
| 7 Timers                | `setTimeout`, `setInterval`, օգտակար է ժամանակի վրա հիմնված ֆունկցիաների համար։               |
| 8 Console               | Debug-ի գործիք։ Դյուրին է, բայց պետք է իմանաս։                                                |
| 9️ OS                    | Օպերացիոն համակարգի մասին ինֆո ստանալու գործիքներ՝ եթե պետք լինի պրոդաքշնում։                 |
| 10 Globals              | Ինչպիսին է `__dirname`, `__filename`, `global`, և այլն։ Օգտագործվում է գրեթե միշտ։            |
